How it calculates SIP

Monthly compounding

Uses monthly rate = (annual rate / 12). Each SIP installment compounds for the remaining months.

Future Value formula

FV ≈ P × [((1+r)^n − 1)/r] × (1+r) where P = monthly SIP, r = monthly return, n = months.

Inflation adjustment

“Real value” = FV / (1 + inflation)^years to show today’s purchasing power.

Host it on GitHub Pages (exact steps)

  1. Create a new repository on GitHub (example name: finlite).
  2. Add 3 files to the repo root: index.html, styles.css, app.js.
  3. Commit the files (via GitHub UI or git).
  4. Go to Settings → Pages.
  5. Under Build and deployment:
    • Source: Deploy from a branch
    • Branch: main / root
  6. Save. GitHub will give you a URL like: https://YOUR-USERNAME.github.io/finlite/
Tip: For a personal homepage, name the repo YOUR-USERNAME.github.io. Then your site becomes https://YOUR-USERNAME.github.io.
